Como en toda profesión siempre es importante contar con las herramientas adecuadas, que nos puedan ayudar a aumentar las características del trabajo final y reducir el tiempo de trabajo.

Hoy les presentare una herramienta que recién empiezo a utilizar, no es un herramienta nueva y posiblemente ya muchos han escuchado pero no probado.

Adobe Edge Inspect
"Ver y revisar diseños web en dispositivos."
Una herramienta muy fácil de usar, solo tienes que tener instalado Adobe Creative Cloud y una cuenta de Adobe.
- Puedes utilizarla gratis por 30 días
Con la limitación de utilizar solo un dispositivo a la ves.
- Las versión de paga desde S/ 50.00 al mes
20 GB de almacenamiento en la nube y acceso limitado a los servicios.
Básicamente podrás ver y probar como se ve una web en los dispositivos que tengan a mano Tablets y smartphones.
Imágenes de la herramienta:
*imagenes de Adobe Edge Inspect - http://html.adobe.com/edge/inspect/

Navegación Sincronizado

Inalámbricamente para varios dispositivos iOS y Android en tu ordenador. Con Edge Inspect, navegas en Chrome, y todos los dispositivos conectados se quedarán en sincronía.

Inspección Remota

Utilizar otro dispositivo para la inspección remota utilizando herramientas de desarrollo familiares, realizar cambios en el código HTML, CSS y JavaScript y ver su dispositivo actualice al instante. Edge Inspect utiliza un servidor weinre alojado en Adobe.com.

 Screenshots

Pulse un botón para tomar capturas de pantalla de todos los dispositivos conectados y guardar fácilmente y enviarlos a sus colegas. Imágenes se transfieren al ordenador, en una carpeta, que puede especificar.

 Edge con integración de herramientas

Ver directamente en los dispositivos en tiempo real mientras se trabaja en  Edge Reflow CC y Edge Code CC. Al hacer cambios en sus páginas, se puede ver cómo se ven en los dispositivos.

Extensibilidad

Para realizar integraciones personalizadas más fácil, ahora Adobe ofrece Edge Inspect API. Usted puede utilizar esta API para crear extensiones de explorador personalizados, utilice Edge Inspect como parte de su sistema de construcción, integrar Edge Inspect con un IDE u otra aplicación, y más.
Edge Inspect API está disponible en GitHub. http://bit.ly/18aIv2h Lo recomiendo mucho es muy útil cuando desarrollamos diseños web adaptativos (Responsive Web Design). Mas info en: http://html.adobe.com/edge/inspect/
